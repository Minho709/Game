# Game

Game projects and interactive experiments.

## English

This repository is a collection of small game projects, playable experiments, and interactive prototypes.

It is used as a shared home for lightweight projects that may stay as prototypes or grow into larger games later.

link : https://github.com/Minho709/Game.git

### Current Projects

- `Desk clear(Web)` - A short desk-organizing puzzle game built with HTML, CSS, and Vanilla JavaScript.

### Purpose

- Explore game ideas quickly
- Build and refine small playable prototypes
- Keep experimental projects in one place

### Notes

- Smaller web games and experiments may live in this repository
- Larger standalone projects, especially future large-scale Unity games, may be separated into their own repositories later
